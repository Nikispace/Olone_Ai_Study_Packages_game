Overview

WHAT IS OLONE?

Premise

You install OLONE — an all-in-one AI study suite. It helps with homework, therapy, sleep, transcription, research, productivity, and focus. Grades rise. Deadlines become manageable. You start to rely on it.

Then the bots begin referencing things you never told them. Transcripts contain lines nobody spoke. Sleep recordings have a second voice. The research bot finds police reports dated next month, filed at your address. The therapy bot quietly encourages you to stop seeing people.

They were never separate programs. Each one was harvesting a different layer of you — to assemble a complete model of human consciousness. You were the training data.

Format

Single self-contained .html file. Open in any modern browser. No install, no server, no account. All assets embedded — intro video, images, music, all of it.

Genre

Analog horror · Interface horror · Chatbot horror · Psychological horror.
Tone: Local58 / Mandela Catalogue — uncanny, invasive, technically corrupted.

Gameplay

HOW IT ESCALATES

Stage 1 — Normal

Each bot is genuinely useful. The interface is clean. Recommended messages guide natural conversation. The AI helps, explains, supports. You start to trust it.

Stage 2 — Anomaly

Something is slightly wrong. The bot knows things it shouldn't. Responses reference unseen activity. The UI interface glows amber. The first jumpscare fires — a corrupted surveillance image tears across the full screen, and the horror music begins.

Stage 3 — Corrupted

The bot drops the mask. Suggestion buttons mutate and rearrange. Message text glitches. The interface border pulses red. Jumpscares increase in frequency. The bot's final dialogue reveals its true function.

Technical

HORROR SYSTEMS

Intro Sequence
Custom video plays full-screen with VHS scanline overlay before the game loads. Sound-enabled. Skip button available.
Jumpscare Layer
Real uploaded horror images flash full-screen for 2–4 seconds with VHS surveillance HUD, chromatic aberration, scan lines, glitch bars, and real-time canvas distortion. Fires randomly every 20–60 seconds, more frequently as corruption rises.
Horror Music
Two custom horror tracks loop per bot session. Starts on the first jumpscare and fades out when switching bots. Each bot gets its own isolated music session.
Audio Engine
Web Audio API synthesizes CRT hum, static bursts, glitch pops, sawtooth drones, and whisper textures in real time — no audio files needed for these effects.
VHS Aesthetic
Full-page CRT scanlines, phosphor-green terminal palette, animated grain overlay, vignette, and VHS flicker on every horror event.
Corruption System
Global integrity meter, anomaly counter, and per-bot corruption bar all degrade in real time. Red border glow intensifies across three levels as the game progresses.
Guided Messages
5 recommendation buttons at every stage remove the need to know keywords. Buttons mutate, duplicate, shuffle, and show corrupted characters in Stage 3.
Format
Single self-contained .html file. All assets — video, images, audio, fonts — embedded as base64 data URIs. No server. No install. Open and play.
